FFRKreeper

Show Instructions
  1. Set up WireShark and your iOS device, using this link
  2. Parse through the messages from ffrk.denagames.com (maybe by IP address) and find your session id (it'll look like this: Cookie: http_session_sid=xxxxxxxxxxxxxxxxxxxxx). Make sure you don't copy the semicolon or the 'http_session_sid' part.
  3. Paste that value in the box above and then go into a battle.
  4. This site will not work for MP Raid Dungeons, but solo Raid Dungeons are fine
  5. In the messages, if you see "..." that means you're either not in a battle or you need to refresh and re-paste your session id.
